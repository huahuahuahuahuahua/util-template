# util-template

使用公共库

模板提供给的命令行地址：

[winkcli-main](https://github.com/huahuahuahuahuahua/winkcli-main/tree/master/)

## 目录结构

```shell

├───.husky
│   └───_
├───src   工具开发
├───README.md 文档
├───README.tmpl.md 公共文档（可生成README.md）
└───test  测试
```

## 示例代码

如需直接使用

```shell
npm run build --文件打包
npm run dev  --监听ts文件
npm run doc --生成文档
npm run lint--测试
npm run release--npm发布
```
